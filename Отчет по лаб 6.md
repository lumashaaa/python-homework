## Ход работы: в ходе  работы были реализованы две версии функции построения бинарного дерева: рекурсивная и нерекурсивная.
Рекурсивная реализация строит дерево путём последовательных рекурсивных вызовов для левого и правого поддерева с уменьшением высоты на каждом шаге.  
Нерекурсивная реализация использует цикл и стек  данных для построения дерева без использования рекурсии.
В корне дерева используется значение root = 13, а значения потомков левый потомок = root + 1, а правый = root - 1.
Для сравнения производительности двух подходов использовался модуль timeit.  
Проводились замерв времени для различных значений высоты дерева, а затем был построен график зависимости времени выполнения от высоты дерева с использованием библиотеки matplotlib.

### Вывод: в результате выполнения лабораторной работы было проведено сравнение рекурсивного и итеративного способов построения бинарного дерева.  
По результатам измерений видно, что нерекурсивная реализация  более стабильна при увеличении высоты дерева, тогда как рекурсивный подход имеет дополнительные расходы, связанные с вызовами функций.  
При построении деревьев большой глубины целесообразнее использовать итеративный алгоритм.
