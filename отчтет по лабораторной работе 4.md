## Ход работы: в лабораторной работе были реализованы две функции вычисления факториала: рекурсивная и итеративная.  
Для сравнения времени работы функций использовался модуль timeit.
Был сформирован фиксированный список чисел, для которых производились замеры времени выполнения.  
Каждое вычисление запускалось несколько раз, чтобы усредненить результат.
Для наглядного сравнения построен график зависимости времени выполнения от входного значения с помощью библиотеки matplotlib.

## Вывод: в результате выполнения работы было проведено сравнение рекурсивного и итеративного способов вычисления факториала.  
По результатам видно, что итеративная реализация работает быстрее и стабильнее, а рекурсивный подход демонстрирует большие расходы.  
Следовательно, при вычислении факториала для больших значений лучше использовать итеративный алгоритм.
