# Отчёт по лабораторной работе №3
## Сукачева Мария Александровна, группа P4150
## Ход работы: в ходе лабораторной работы была реализована рекурсивная функция генерации бинарного дерева `gen_bin_tree`.  
Дерево представлено в виде словаря, который содержит значение текущего узла и ссылки на левое и правое поддеревья.
В корне дерева размещается значение root, высота дерева определяется параметром height.  
Для root = 13 значение левого потомка - root + 1, а для правого потомка - root-1.  
Функция рекурсивно строит дерево, уменьшая значение высоты на каждом шаге.
Для проверки корректности работы функции были реализованы тесты с использованием библиотеки unittest.
## Вывод
В результате выполнения лабораторной работы была реализована рекурсивная генерация бинарного дерева заданной высоты.
