# Лабораторные работы №7–9  
**Клиент-серверное приложение для работы с курсами валют ЦБ РФ**  
##Студентка:Сукачева Мария, группа P4150  

### Лабораторная №7 — Декоратор логирования
- Реализован универсальный параметризованный декоратор `@logger(handle=…)`
- Поддерживает `sys.stdout`, `io.StringIO`, `logging.Logger`
- Логирует начало/окончание вызова, аргументы, результат, ошибки
- Функция `get_currencies()` полностью покрыта логированием и тестами
- Исключения (ConnectionError, ValueError, KeyError и др.) пробрасширения корректно обрабатываются

### Лабораторная №8 — Веб-приложение
- Полностью реализована архитектура MVC
- Маршруты: `/`, `/users`, `/user?id=…`, `/currencies`
- Шаблонизатор Jinja2 с `PackageLoader`
- Модели с геттерами/сеттерами в пакете `models`
- Функция `get_currencies` из лабы 7 интегрирована и используется

### Лабораторная №9
- База данных `:memory:` с таблицей `currency`
- Реализован полный CRUD:
  - **Create/Update** — кнопка «Обновить из ЦБ» (UPSERT по `char_code`)
  - **Read** — таблица на странице `/currencies`
  - **Delete** — кнопка «Удалить» у каждой валюты
- Все запросы параметризованные (защита от SQL-инъекций)
- База создаётся один раз при старте приложения
- Данные сохраняются между запросами
