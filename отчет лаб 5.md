## Ход работы: в ходе лабораторной работы была реализована нерекурсивная функция построения бинарного дерева gen_bin_tree.  
Дерево представляется в виде словаря, который содержит значение узла и ссылки на потомков.
Построение дерева осуществляется с использованием очереди deque из модуля collections. Это позволяет реализовать обход дерева по уровням без использования рекурсии.  
В очередь помещаются текущий узел и уровень его глубины, что даёт возможность контролировать высоту дерева.
Значения левого и правого потомков вычисляются с помощью параметризованных функций left_branch и right_branch, передаваемых в качестве аргументов функции.  
По умолчанию используются лямбда-функции, соответствующие root = 13, left = root + 1, right = root - 1.
Корректность работы проверяется с помощью модульных тестов, которые реализовывались с помощью библиотеки unittest. Они проверяют значение корня и корректность построения уровней дерева.

## Вывод: в результате выполнения лабораторной работы была реализована нерекурсивная генерация бинарного дерева с использованием очереди.  
Полученная реализация формирует структуру дерева заданной высоты, поддерживает параметризацию и проходит модульные тесты.
